我们的代码里使用了 localStorage（浏览器本地存储技术）。这相当于在你的手机浏览器里借用了一块“小 U 盘”，专门用来存放你这个决定器的数据。你今天关掉网页、杀掉浏览器后台，明天再打开，数据和图片都会乖乖待在那里。

但是，既然是“本地存储”，在手机上使用时就有几个容易导致数据丢失的雷区，你需要特别注意：

清理手机缓存/浏览器数据：如果你在手机设置里清理了浏览器的缓存，或者使用了清理软件，这些外卖数据就会被一起清空。

无痕模式/隐私模式：如果你是在浏览器的“无痕模式”下打开的这个网页，只要一关掉页面，数据就会立刻灰飞烟灭。

微信/QQ 内置浏览器（重点注意）：如果你是把链接发在微信里，直接在微信里点开用的，微信系统有时会不定时自动清理内置浏览器的缓存文件。

强烈建议的用法：在微信里点开后，点击右上角的三个点，选择“在浏览器打开”（苹果手机用 Safari，安卓用 Chrome 或自带浏览器）。然后在使用系统浏览器时，把它“添加到主屏幕”。这样它就会像一个独立的 App 一样留在你的手机桌面上，数据最稳定！

不互通：你在 Safari 浏览器里存的数据，如果明天换成用 Chrome 浏览器打开，是看不到的。数据是跟着具体哪个浏览器走的。
